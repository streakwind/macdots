# mac
switched from arch to mac for a variety of reasons. not much to config (imo) because I really only config'd hyprland and the terminal.

brought over the old .zshrc, made ghostty and nvim configs

just like before, the `oh-my-posh` themes are from https://github.com/JanDeDobbeleer/oh-my-posh/tree/main/themes

## installation (mac)
install nvim, ghostty, oh-my-posh, stow

then, after cloning the repo, run
```
stow configs
```
in the repo root dir
